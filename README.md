# 🧮 BMI Calculator

A simple and responsive **BMI (Body Mass Index) Calculator** built using **HTML**, **CSS**, and **JavaScript**. Users can enter their height and weight to calculate their BMI and determine their weight category.

## 🚀 Features

* 📏 Enter height and weight
* ⚡ Instantly calculate BMI
* 📊 Displays BMI value
* 🩺 Shows BMI category (Underweight, Normal, Overweight, or Obese)
* 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
